## androidx-preference

Eclipse library project based on:<br/>
https://maven.google.com/androidx/preference/preference/1.1.0/preference-1.1.0.aar

**Import:**
- _File > Import... > Team > Team Project Set > URL:_<br/>
  https://raw.githubusercontent.com/dandar3/android-androidx-preference/1.1.0/.projectset

**Requires:**
- `Android 9 (API 28) SDK Platform`
